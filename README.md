# Hangman-in-Java
A small, fun, ascii-based command line hangman game in Java!  
Use any words file you want to play with custom words from a movie, book, show or perhaps you want to test your knowledge of words in another language.  
This hangman game is fun for the whole family, all you need is a terminal, Java and half a brain.  
An ascii stickman will show you how close you are to death! Have fun!
